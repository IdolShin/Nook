# Nook Logo — Asset Pack

## What's included

### SVG (벡터, 무한 확대 가능 — 웹/인쇄 모두 권장)
- `nook-mark-primary.svg` — 그라디언트 배경 (메인)
- `nook-mark-on-white.svg` — 흰 배경용 그린 스트로크
- `nook-mark-mono-dark.svg` — 흑백 다크
- `nook-mark-mono-light.svg` — 흑백 라이트 (어두운 배경)
- `nook-lockup-*.svg` — 로고 + Nook 워드마크 (4 variants)

### PNG (래스터, 사이즈별)
- 마크: 256 / 512 / 1024 / 2048 px
- Lockup: 512 / 1024 / 2048 폭
- 4가지 컬러 변형 각각

### Favicon / 앱 아이콘
- `favicon-16.png` ~ `favicon-128.png`
- `apple-touch-icon.png` (180×180)

### 소셜 공유
- `nook-og-image.png` — 1200×630, OG/Twitter Card용

## 어디에 뭘 쓰나

| 용도 | 파일 |
|---|---|
| 웹사이트 헤더 | `nook-lockup-primary.svg` |
| 다크모드 헤더 | `nook-lockup-mono-light.svg` |
| 명함·인쇄 | SVG (어떤 사이즈든) |
| 인스타 프로필 | `nook-mark-primary-1024.png` |
| 앱 아이콘 (iOS/Android) | `nook-mark-primary-2048.png` |
| 카톡/페북 공유 미리보기 | `nook-og-image.png` |
| 브라우저 탭 | `favicon-32.png` |
| 아이폰 홈화면 | `apple-touch-icon.png` |

## 색상 사양
- Primary green gradient: `#22B889` → `#0F6B4D`
- Solid green: `#1D9E75`
- Dark: `#1A1A1F`
- 항상 sharp corners 8px radius 유지
